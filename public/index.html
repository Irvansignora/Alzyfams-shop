Pertanyaan?

Apa yang dimaksud "free setup"? +
Kami lakukan setup server/hosting, deploy aplikasi, konfigurasi database, dan pastikan website sudah berjalan. Anda tidak perlu tahu coding sama sekali — tinggal terima jadi.
Kasir POS harga Rp 50k per user artinya apa? +
Setiap akun pengguna (kasir atau admin) dikenakan biaya Rp 50.000 sekali bayar. Jika kamu perlu 3 kasir + 1 admin = 4 user × Rp 50k = Rp 200k. Akses selamanya tanpa biaya bulanan.
Apakah Kasir POS bisa dipakai tanpa internet? +
Ya! Kasir POS kami bersifat offline-first. Semua data transaksi disimpan di perangkat. Anda bisa bertransaksi meski tidak ada koneksi internet.
Berapa lama proses setup setelah bayar? +
Mulai proses dalam 1×24 jam setelah pembayaran terkonfirmasi. Rata-rata selesai 4–8 jam kerja, tergantung konfigurasi yang diminta.
Ada garansi atau support setelah setup? +
Support via WhatsApp 30 hari (60 hari untuk bundle). Jika ada bug atau masalah teknis setelah setup, kami bantu perbaiki tanpa biaya tambahan.
